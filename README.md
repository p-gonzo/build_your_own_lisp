## Setup: 

Ensure you have c and lldb / gdb installed on your machine.

- `which lldb`
- `which cc`

## Compilation:

- `cc -g --std=c99 -Wall hello_world.c -o hello_world`
- `./hello_world`

## Debugging:

- `lldb hello_world`
